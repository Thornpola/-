## 📝 Add Your Content

### 1. Add Images
Place your portfolio images in:
```
d:\wep\library\public\images\
```
Example: `Jpg.jpg`, `2.jpg`, `3.jpg`, `photo_2026-01-26_12-43-09.jpg`

### 2. Add PDFs
Place your PDF files in:
```
d:\wep\library\public\pdfs\
```
Example: `SPSS.pdf`, `RCBD.pdf`, `RCB.pdf`

### 3. Update Portfolio Items
Edit `server.py` - update `PORTFOLIO_DATA` with your content:
```python
PORTFOLIO_DATA = [
    {
        "id": 0,
        "title": "Your Title",
        "description": "Your description",
        "image": "/images/your-image.jpg",
        "pdf": "/pdfs/your-file.pdf"
    },
    ...
]
```

## 🔧 Configuration

### Change Port
Edit `.env`:
```
PORT=3000  ← Change this
```

### Production URL
Edit `.env`:
```
PROD_URL=https://your-domain.com
PROD_API_URL=https://your-domain.com/api
```

### Contact Email
Edit `server.py`:
```python
'contactEmail': 'your-email@example.com',  ← Update this
```

## 📡 API Endpoints

### GET /api/config
Returns server configuration:
```json
{
  "environment": "development",
  "baseUrl": "http://localhost:3000",
  "apiUrl": "http://localhost:3000/api",
  "contactEmail": "thornpola6@gmail.com",
  "phone": "+855 88 455 4566"
}
```

### GET /api/portfolio
Returns portfolio items with images and PDFs.

### POST /api/contact
Submit contact form:
```json
{
  "name": "John Doe",
  "email": "john@example.com",
  "subject": "Inquiry",
  "message": "Message content"
}
```

## 💡 Tips

1. **Images Load:** Place images in `public/images/` - they'll be served at `/images/name.jpg`
2. **PDFs Work:** Place PDFs in `public/pdfs/` - they'll be served at `/pdfs/name.pdf`
3. **API Test:** Visit http://localhost:3000/api/config to test the API
4. **Console Logs:** Check terminal for debug messages and contact submissions
5. **Development Mode:** Server auto-reloads on file changes
